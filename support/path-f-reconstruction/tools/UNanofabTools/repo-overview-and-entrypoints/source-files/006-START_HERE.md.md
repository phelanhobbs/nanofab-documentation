## 📝 Quick Configuration (.env file)

Open `.env` and update these 5 settings:

1. **SECRET_KEY** - Generate one:
   ```bash
   python3 -c 'import secrets; print(secrets.token_hex(32))'
   ```

2. **DEBUG_MODE** - Set to `True` for testing:
   ```
   DEBUG_MODE=True
   ```

3. **HOST** - Use localhost for development:
   ```
   HOST=localhost
   ```

4. **SSL Paths** - Point to your certificates:
   ```
   SSL_CERT_PATH=/path/to/cert.pem
   SSL_KEY_PATH=/path/to/key.pem
   ```

5. **Duo Credentials** - Add your Duo keys:
   ```
   DUO_IKEY=your-key
   DUO_SKEY=your-secret
   DUO_HOST=your-host
   ```
